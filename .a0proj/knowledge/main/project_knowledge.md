# Salon Flow - Project Knowledge Base

## Project Overview

**Salon Flow** is an AI-powered, event-driven, multi-tenant Salon Management SaaS platform designed to replace ReSpark for Jawed Habib Hair & Beauty Salon franchise in Kurnool, India.

## Architecture

### 4 React PWAs
1. **Owner PWA** - Analytics, multi-location, financial reports
2. **Manager PWA** - Operations, staff management, inventory
3. **Staff PWA** - Schedule, service delivery, upsell
4. **Client PWA** - Conversational UI, booking, loyalty

### 3 Consolidated Services
1. **API Service** - All business logic
2. **AI Agent Service** - Chat, voice, marketing agents
3. **Notification Service** - WhatsApp, push, email

### AI Agents
1. **Booking Agent** - Appointment management
2. **Support Agent** - Customer queries
3. **Marketing Agent** - Campaigns, offers
4. **Analytics Agent** - Insights, reports
